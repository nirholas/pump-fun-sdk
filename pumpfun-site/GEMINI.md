# Solana Launchpad UI — Gemini Instructions

> Static token launchpad UI template. Zero dependencies. Pure HTML/CSS/JS.

## Key Facts

- 4 pages: Board, Create, Trade, Profile
- Dark theme with neon green accent (`#7bff69`)
- All data is mock — no real API calls
- CSS custom properties for theming — see `:root` in `styles.css`
- Mobile-first responsive design
- Zero npm dependencies — vanilla only

## Do Not

- Add npm packages or CDN scripts
- Use frameworks (React, Vue, etc.)
- Hardcode hex colors (use CSS variables)
- Make real API calls (mock data only)
- Write desktop-first CSS
